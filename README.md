## 🧑‍💻 Running Locally

Clone the repository:

```bash
git clone https://github.com/Areeba-Khaliq/Nexium_Areeba_assign1.git
Install dependencies:
pnpm install
Run the development server:
pnpm dev
Visit http://localhost:3000 to view your app.

```
